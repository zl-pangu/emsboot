关于echarts

ECharts (Enterprise Charts 商业产品图表库)

提供商业产品常用图表，底层基于ZRender（一个全新的轻量级canvas类库），创建了坐标系，图例，提示，工具箱等基础组件，并在此上构建出折线图、柱状图、散点图、K线图、饼图、雷达图、地图、和弦图、力导向布局图、仪表盘以及漏斗图，同时支持任意维度的堆积和多图表混合展现。

标签式单文件引用echarts

1、新建一个echarts.html文件。为ECharts准备一个具备大小（宽高）的Dom。

2、引入echarts-plain.js或者 echarts-plain-map.js

3、引入主题文件theme.js

4、<script>标签内动态加载echarts和所需图表，初始化时加载所需主题，包括flat主题，blue主题，除地图外建议使用flat主题。缺省时为默认主题。回调函数中可以初始化图表并驱动图表的生成，option见API & Doc
